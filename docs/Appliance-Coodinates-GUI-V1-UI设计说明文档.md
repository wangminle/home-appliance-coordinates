# 家居设备坐标距离角度绘制工具 - UI设计说明文档

## 1. 设计概述

### 1.1 设计目标
创建一个专业、直观、美观的桌面应用界面，为用户提供优秀的家居设备布局规划体验。

### 1.2 设计原则
- **功能性优先**: 界面服务于功能，不做无意义的装饰
- **直观易用**: 操作流程符合用户直觉，学习成本低
- **视觉协调**: 配色、字体、布局保持一致性
- **响应及时**: 界面反馈及时，状态变化清晰

### 1.3 目标用户
- 智能家居规划师
- 家居装修设计师
- IoT设备布局工程师
- 家庭用户

## 2. 整体布局设计

### 2.1 窗口规格
```
窗口尺寸: 1280x800 像素
布局方式: 左右分割式
左侧区域: 800x800 像素 (可视化展示)
右侧区域: 480x800 像素 (功能操作)
窗口属性: 固定大小，不可调整，居中显示
```

## 3. 左侧画布区域设计

### 3.1 Canvas基础设计
```
尺寸: 800x800 像素
背景色: #e0f7fa (浅蓝色)
边框: 1px solid #b0bec5
内边距: 40px (为坐标轴标签预留空间)
绘制区域: 720x720 像素
```

### 3.2 坐标系统视觉设计

#### 3.2.1 网格系统
- **网格线颜色**: `#b0bec5` (灰蓝色)
- **网格线宽度**: 0.5px
- **网格间距**: 动态计算，保证整数刻度
- **网格类型**: 正方形网格，X/Y轴等间距

#### 3.2.2 坐标轴设计
- **轴线颜色**: `#37474f` (深灰色)
- **轴线宽度**: 1.5px
- **X轴**: 水平居中，贯穿整个绘制区域
- **Y轴**: 垂直居中，贯穿整个绘制区域
- **原点标记**: 蓝色圆点，半径4px，颜色`#1e88e5`

#### 3.2.3 刻度标签
- **字体**: Arial, 10px
- **颜色**: `#37474f`
- **位置**: 
  - X轴标签: 轴线下方5px
  - Y轴标签: 轴线左侧5px
- **格式**: 整数显示，范围内均匀分布

### 3.3 设备点显示设计

#### 3.3.1 设备标记
- **形状**: 圆形
- **颜色**: `#c62828` (红色)
- **大小**: 半径2.5px
- **描边**: 无描边
- **层级**: 在网格之上，标签之下

#### 3.3.2 设备标签框
- **背景色**: `#ffffe0` (浅黄色)
- **边框**: 2px solid `#d32f2f` (红色)
- **阴影**: 2px 2px 5px rgba(0,0,0,0.4)
- **圆角**: 无圆角，矩形设计
- **尺寸**: 90px × 35px
- **字体**: Arial, 16px, bold
- **文字颜色**: `#c62828`
- **文字对齐**: 居中对齐

#### 3.3.3 坐标信息显示
- **位置**: 标签框正下方，间距12px
- **字体**: Arial, 14px, bold
- **颜色**: `#c62828`
- **内容**: 
  - 第一行: `X: [坐标值]`
  - 第二行: `Y: [坐标值]`
- **行间距**: 18px

#### 3.3.4 标签定位逻辑
- **左侧定位**: 标签框右边缘距离设备点10px
- **右侧定位**: 标签框左边缘距离设备点10px
- **智能避让**: 自动检测边界，避免标签溢出

### 3.4 交互测量设计

#### 3.4.1 十字光标
- **颜色**: `rgba(0, 0, 0, 0.5)` (半透明黑色)
- **线宽**: 0.5px
- **范围**: 贯穿整个绘制区域
- **显示条件**: 鼠标在Canvas区域内

#### 3.4.2 测量点标记
- **形状**: 圆形
- **颜色**: `#2e7d32` (绿色)
- **大小**: 半径4px
- **层级**: 最上层显示

#### 3.4.3 测量连线
- **颜色**: `#4caf50` (绿色)
- **线宽**: 1.5px
- **样式**: 实线
- **起点**: 坐标原点(0,0)
- **终点**: 测量点位置

#### 3.4.4 测量信息框
- **背景色**: `rgba(255, 255, 255, 0.85)` (半透明白色)
- **边框**: 1px solid `#2e7d32` (绿色)
- **位置**: 测量点右下方，偏移15px
- **内边距**: 8px
- **字体**: Arial, 12px
- **文字颜色**: `#1b5e20` (深绿色)
- **内容格式**:
  ```
  坐标: X: [值], Y: [值]
  距离: [值]
  角度: [值]°
  ```

## 5. 颜色系统设计

### 5.1 主色调
```css
/* 主要蓝色系 */
--primary-blue: #2196f3;
--light-blue: #e3f2fd;
--canvas-bg: #e0f7fa;

/* 功能色彩 */
--success-green: #4caf50;
--warning-orange: #ff9800;
--error-red: #f44336;
--device-red: #c62828;

/* 中性色彩 */
--dark-gray: #37474f;
--medium-gray: #666666;
--light-gray: #f5f5f5;
--border-gray: #e0e0e0;
--white: #ffffff;
```

### 5.2 色彩语义
- **蓝色系**: 主要操作、链接、选中状态
- **绿色系**: 成功操作、添加操作、测量功能
- **红色系**: 删除操作、错误状态、设备标记
- **橙色系**: 警告操作、重置功能
- **灰色系**: 文字、边框、背景

### 5.3 色彩对比度
- 所有文字与背景对比度 ≥ 4.5:1
- 重要按钮与背景对比度 ≥ 3:1
- 确保色盲用户可正常使用

## 7. 交互设计规范

### 7.1 鼠标交互

#### 7.1.1 Canvas区域
- **鼠标移动**: 显示十字光标，实时跟踪
- **左键单击**: 创建测量点，显示测量信息
- **右键单击**: 清除测量点和信息
- **鼠标离开**: 隐藏十字光标

#### 7.1.2 设备列表
- **单击选择**: 高亮选中行，填充输入框
- **双击编辑**: 直接进入编辑模式
- **鼠标悬停**: 行背景变色提示

#### 7.1.3 按钮交互
- **鼠标悬停**: 背景色变亮，显示手型光标
- **按下**: 轻微阴影内陷效果
- **禁用**: 灰色显示，禁止点击

### 7.2 键盘交互

#### 7.2.1 快捷键支持
- **Ctrl+S**: 导出PNG
- **Ctrl+R**: 重置数据
- **Ctrl+A**: 添加设备
- **Delete**: 删除选中设备
- **Enter**: 确认当前操作

#### 7.2.2 Tab键导航
- 按逻辑顺序在可聚焦元素间切换
- 焦点状态有明显视觉反馈
- 支持Shift+Tab反向导航

### 7.3 状态反馈

#### 7.3.1 加载状态
- 长时间操作显示进度条
- 按钮操作显示加载动画
- 导出功能显示进度提示

#### 7.3.2 错误状态
- 输入验证错误：红色边框 + 错误提示
- 操作失败：弹出错误对话框
- 网络错误：友好的错误信息

#### 7.3.3 成功状态
- 操作成功：绿色提示信息
- 数据保存：临时成功提示
- 导出完成：文件位置提示
